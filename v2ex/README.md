# **v2ex**

### 功能：

模拟登录v2ex完成任务领钱 OvO

### 依赖：

*  Requests

*  Lxml

### 如何使用：

在当前目录下新建文件`v2ex_config.py`, 像这样：

> USER = 'xxxx'

> PASSWD = 'xxxx'

然后`$python v2ex.py` 就可以了

你可以设置开机启动或者定时任务什么的 XD
